# CodeX プロジェクトページ草案

## プロジェクト概要
- **タイトル**: Town Without Light / 灯りのない町
- **タグライン**: 霧と闇に飲み込まれた町で、唯一の光を頼りに兄を探す短編ホラー。
- **ジャンル**: 一人称探索ホラー、ストーリードリブン、シングルプレイ
- **ターゲットプラットフォーム**: Steam (Windows)
- **開発期間**: 6か月
- **目標**: 初月売上5〜10万円、レビュー「やや好評」以上

## ページ構成
1. **ヒーローヘッダー**: キービジュアル＋タグライン＋CTA（進捗更新をフォロー）
2. **プロジェクト詳細**
   - ストーリー概要
   - コアメカニクス（光管理、探索、パズル）
   - スクリーンショット（ブロックアウト段階→完成まで並べて開発進捗を見せる）
3. **進捗タイムライン**: フェーズ0〜7を1行説明で表示
4. **マイルストーン**
   - プロトタイプ公開
   - フルマップ完成
   - ストアページ公開
   - リリース
5. **サポートのお願い**
   - フィードバックフォームリンク
   - プレイテスター募集
6. **チーム/個人情報**: ソロ開発者としての経歴＋過去作リンク

## 更新方針
- 週1回、スクリーンショット＋短い進捗メモを投稿
- 主要マイルストーンでは長文記事を投稿
- QA期間にはテスト募集告知をピン留め

## 必要アセット
- 1200x675pxのカバー画像（工場の煙突＋懐中電灯の光）
- 3枚のワークインプログレス画像
- ロゴ（英字＋和字）
- プロジェクトカラー: #0b132b / #ffdda1
