# コンセプト要約（1ページ）
- **タイトル（仮）**: 灯りのない町（Town Without Light）
- **ジャンル**: 一人称探索型ホラー（短編）
- **ターゲット**: ソロで没入型ホラーを好むPCゲーマー（Steam / Windows）
- **プレイ時間**: 60〜90分、リプレイ性はエンディング分岐で確保
- **コア体験**: 無灯の町で唯一の光源である懐中電灯を頼りに、空洞化した生活の痕跡と不可視の存在を追う「孤独な調査」
- **差別化要素**: 視界を奪う漆黒の霧、音で導く探索、町全体が巨大なパズルのように段階開放される構造
- **制作目標**: 6か月以内にSteamでリリースし、初月売上5〜10万円を達成する。

# 世界観と舞台設定
- **舞台**: 北国の鉱山町。慢性的な停電と住民失踪で行政からも放棄され、夜になると町全体が闇に沈む。
- **時代**: 現代。異常気象による雪混じりの霧が常時立ち込める。
- **プレイヤー**: 行方不明の兄を捜索するフリーライター。兄は町に残された最後の住人と言われる。
- **キー要素**: 廃工場、閉ざされた学校、町役場、防災無線塔、地下シェルター。電力は断続的で、音声ログやメモが主要な手掛かりとなる。
- **テーマ**: 「消えゆくコミュニティ」と「光の希少性」。誰もいない町を歩く孤独感と、人工的な光や音が持つ温かさの対比。

# 簡易GDD
## ゲームピラー
1. **光と闇の管理**: 懐中電灯のバッテリー、発電機、信号灯など光を巡る意思決定。
2. **環境ストーリーテリング**: 空き家の生活感、メモ、録音、町内放送で断片を集める。
3. **緊張と静寂のリズム**: 敵の直接的な脅威は少なく、幻聴や環境変化で心理的恐怖を演出。

## ゲーム仕様概要
| 項目 | 内容 |
| --- | --- |
| 視点 | 一人称（マウス＋キーボード） |
| コントロール | WASD移動、Shiftダッシュ、E調査、F懐中電灯、タブ手帳 |
| 進行 | 手がかり収集→エリア解放→主要施設調査→エンディング |
| 失敗条件 | 闇に包まれる（光源切れ＆霧濃度MAX）／特定イベントでの判断ミス |
| UI | 簡易手帳、バッテリーゲージ、現在の目的 |
| サウンド | ローファイBGM＋環境音。重要情報はラジオ音声やノイズに隠す |
| ローカライズ | 日本語／英語対応。UIテキストはScriptableObjectで管理 |

## コンテンツ構成
- **主要施設**: 中心街（商店街＋役場）、住宅街、工場、学校、地下シェルター。
- **敵/脅威**: 黒霧（視界阻害）、幻影（視覚演出）、音声誘導。
- **収集物**: 録音テープ、メモ、写真、兄のノート断片。
- **パズル**: 発電ルート切り替え、無線周波数合わせ、シェルター暗号。

# ゲームループ
1. 目的の提示（手帳更新）
2. 懐中電灯のバッテリーと道具を整える
3. 闇に覆われた区画を探索し、手がかりやパズルを発見
4. 発電機・スイッチを操作して次のエリアを開放
5. 兄の足取りに近づくにつれ霧が濃くなり、再び目的更新
6. エンディング条件を満たすと分岐イベントへ

# マップ構成ラフ
- **全体**: 放射状の町。中心街（役場・防災無線塔）を中央に、東に工場、西に学校、南北に住宅街。
- **導線**: 初期は中心街と南住宅のみ開放。発電ルートを繋ぐことで北住宅→学校→工場→地下シェルターの順に解放。
- **ランドマーク**: 無線塔の赤い光、工場の煙突、学校の時計台で方向感を与える。
- **隠しルート**: 裏路地の梯子、地下配管を通るショートカット。

# エンディング分岐案
1. **再会エンド（TRUE）**: 兄を救出し町を離れる。条件：すべての手がかりを集め、シェルターの暗号を解く。
2. **消失エンド**: 光源が失われ霧に飲まれる。条件：バッテリー管理に失敗し闇イベントをトリガー。
3. **代替エンド**: 兄の代わりにプレイヤーが町の「灯台守」として残る。条件：町の発電網を完全修復し無線塔を稼働。
