# Town Without Light

半年でSteam向け短編ホラーゲームを制作し、初月売上5〜10万円を目指す計画のドキュメント置き場です。

## プロジェクトゴール
1. **フェーズ0（企画準備）**で世界観、ゲーム性、アセット要件を固める。
2. **フェーズ1〜3**でプロトタイプからゲーム進行までを作り切る。
3. **フェーズ4〜6**で演出、QA、Steam公開準備を完了する。
4. **フェーズ7**でリリース後の運用と改善を継続する。

詳細は各ドキュメントを参照してください。
